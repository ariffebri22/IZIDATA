## Installation

Install API with npm :

-   Clone the Repo

```
  git clone https://github.com/ariffebri22/IZIDATA.git
```

-   Go To Folder Repo

```
  cd IZIDATA
```

-   Install Module

```
  npm install
```

-   Setup .env

```
DB_HOST=
DB_USER=
DB_PASSWORD=
DB_NAME=
DB_PORT=

SECRET_KEY=
```

-   Type `npm start` To Start Production
-   To fill in the env, you can contact me at [Contact](#contact)
